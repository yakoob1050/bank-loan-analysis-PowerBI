# bank-loan-analysis-PowerBI
Project Overview

This project presents a complete analytical overview of bank loan data using Power BI. It helps monitor loan applications, funded amounts, repayments, borrower financial health, regional lending patterns, and loan performance.
The dashboard is divided into three pages:

KPI Performance Dashboard

Good Loan and Loan Status Dashboard

Details Dashboard

PAGE 1 – KPI Performance Dashboard
Objective

To present key loan performance metrics, including totals, Month-to-Date (MTD) values, and Month-over-Month (MoM) trends.

Key Performance Indicators
Total Loan Applications

Overall number of loan applications with MTD and MoM breakdown.

Total Funded Amount

Total amount disbursed as loans with MTD and MoM analysis.

Total Amount Received

Total repayment collected from borrowers with MTD and MoM tracking.

Average Interest Rate

Average interest rate across all loans with MTD and MoM variation.

Average Debt-to-Income Ratio (DTI)

Average borrower DTI with MTD and MoM changes to assess financial health.

PAGE 2 – Good Loan Analysis and Loan Status Grid
Objective

To evaluate loan quality through Good Loan metrics and provide a grid-based performance summary categorized by loan status.

Good Loan KPIs

Good Loan Application Percentage

Good Loan Applications

Good Loan Funded Amount

Good Loan Total Received Amount

Loan Status Grid

A structured table containing:

Total Loan Applications

Total Funded Amount

Total Amount Received

MTD Funded Amount

MTD Amount Received

Average Interest Rate

Average DTI

Charts on This Page

Monthly Trends by Issue Date (Line Chart)

Regional Analysis by State (Filled Map)

Loan Term Distribution (Donut Chart)

Employment Length Analysis (Bar Chart)

Loan Purpose Breakdown (Bar Chart)

Home Ownership Analysis (Tree Map)

PAGE 3 – Details Dashboard
Objective

To provide a detailed, consolidated, and user-friendly summary of essential loan-related information.

Purpose

This dashboard offers a single page where users can view:

Individual loan details

Borrower profiles

Loan performance information

It serves as a central reference for deep loan-level analysis.

Tools and Techniques Used

Power BI Desktop

Power Query

DAX for KPI calculations (MTD, MoM, Good Loan %, DTI, Interest Rate)

Data modeling and relationships

Interactive slicers

Various visual types: line charts, bar charts, donut charts, maps, treemaps, and grids
