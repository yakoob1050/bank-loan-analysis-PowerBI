# bank-loan-analysis-PowerBI
**Project Overview**
This project presents a complete analytical overview of bank loan data using Power BI. It helps monitor loan applications, funded amounts, repayments, borrower financial health, regional lending patterns, and loan performance.

**The dashboard is divided into three pages:**

1. KPI Performance Dashboard
2. Good Loan and Loan Status Dashboard
3. Details Dashboard
**PAGE 1 – KPI Performance Dashboard**
    **Objective**
To present key loan performance metrics, including totals, Month-to-Date (MTD) values, and Month-over-Month (MoM) trends.
--Key Performance Indicators--
1. Total Loan Applications

2. Overall number of loan applications with MTD and MoM breakdown.

3. Total Funded Amount

4. Total amount disbursed as loans with MTD and MoM analysis.

5. Total Amount Received

6. Total repayment collected from borrowers with MTD and MoM tracking.

7. Average Interest Rate

8. Average interest rate across all loans with MTD and MoM variation.

9. Average Debt-to-Income Ratio (DTI)

10. Average borrower DTI with MTD and MoM changes to assess financial health.

**PAGE 2 – Good Loan Analysis and Loan Status Grid**
**Objective**
To evaluate loan quality through Good Loan metrics and provide a grid-based performance summary categorized by loan status.
1. Good Loan KPIs
2. Good Loan Application Percentage
3. Good Loan Applications
4. Good Loan Funded Amount
5. Good Loan Total Received Amount
6. Loan Status Grid



**PAGE 3 – Details Dashboard**
**Objective**
To provide a detailed, consolidated, and user-friendly summary of essential loan-related information.
**Purpose**
This dashboard offers a single page where users can view:
Individual loan details
Borrower profiles
Loan performance information
It serves as a central reference for deep loan-level analysis.

**Tools and Techniques Used**
1. Power BI Desktop
2. Power Query
3. DAX for KPI calculations (MTD, MoM, Good Loan %, DTI, Interest Rate)
4. Data modeling and relationships
5. Interactive slicers
6. Various visual types: line charts, bar charts, donut charts, maps, treemaps, and grids
7. Project Structure
8. Bank-Loan-Analysis
9. Dataset
10. PowerBI File (PBIX)
11. Dashboard Screenshots

**Conclusion**

The Bank Loan Analysis Dashboard provides comprehensive visibility into the loan portfolio. It helps analyze performance, evaluate borrower financial behavior, and identify trends across regions and loan types. This project demonstrates Power BI skills in data modeling, DAX, dashboard design, and financial analysis.
